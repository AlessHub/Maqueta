Este proyecto trata sobre hacer una maqueta de un mockup anterior, el cual estaba basado en una página web de viajes.

El proyecto se ha hecho en Visual Studio Code con html, css y flexbox, con ayuda de imagenes de internet para rellenar huecos y iconos. Y con la ayuda de google para buscar soluciones a diversos problemas,
como por ejemplo hacer navbars o dropdown. También se ha usado para buscar otras páginas web como inspiración y ver sus soluciones a nuestros mismos problemas.

El proceso en si empezó con html y css, y a continuacion del proyecto he ido por ambas partes poco a poco, viendo como reaccionaba el codigo a distintos elementos, tags etc. Ademas de como dicho anteriormente,
usando mucho google para ver distintas maneras de solucionar problemas con el código.

Empezando con el header usé flexbox desde el comienzo para hacer la primera navbar, a lo largo del proyecto tuve que rehacer el header varias veces para poder implementar nuevos elementos,
o hacer que se vea mejor tanto como en la página web como en el código mismo.

Despues de acabar con el header seguí con el segundo navbar, el cual, al igual que el anterior, tuve que rehacer varias veces por los mismos motivos.

Una vez terminado el segundo navbar, empezé con el apartado main, y sus imagenes. En esta parté aprendí bastante sobre flexbox, al intentar poner imagenes flotando a la derecha del texto. Estuve bastante rato,
pero al final con la ayuda de google conseguí hacerlo, aprendiendo la importancia del orden dentro del código, sobretodo a la hora de usar flexbox.

Cuando terminé de ordenar el main con sus imagenes, seguí con la siguiente página, destinations. Pensé que sería mas fácil pero tardé bastante mas tiempo del que me hubiese gustado. Acabe aprendiendo que a la
hora de usar flexbox, usar containers hace todo mucho más fácil.

Las dos siguientes páginas fueron mucho más fáciles, y no tardé mucho en hacerlas, tiempo el cual utilizé en intentar hacer un boton de lenguaje con dropdown. Despues de varias horas intentando seguir un
tutorial tuve que rehacer el boton siguiendo otra página, el cual acabó funcionando y pude adaptar a la página, aun que me hubiese gustado añadir un par de cosas como por ejemplo las banderas con sus respectivos
paises, en general acabé bastante contento con el resultado tanto de el boton de dropdown como de la página web en general.
